## Install

- Install theme
    - Download .vsix file from the release page
    - Go to extensions
    - Click the three dots, and install from VSIX 
    - Install nso-theme.0.0.1.vsix
- Install font
    - Pixel Code: https://github.com/qwerasd205/PixelCode
